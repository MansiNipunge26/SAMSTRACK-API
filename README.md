# SAMSTRACK API

SAMSTRACK API is a Spring Boot based RESTful application developed to manage and track system or asset-related data efficiently.

## Technologies Used

* Java
* Spring Boot
* Maven
* REST API
* MySQL (if used)

## Features

* Create and manage records
* Update and delete data
* RESTful endpoints
* Maven build support

## Project Structure

* `src/main/java` – Application source code
* `pom.xml` – Maven dependencies
* `mvnw` – Maven wrapper

## How to Run

1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Run the main class
4. Access APIs on:
   `http://localhost:8080`
